# Keylogger

## Description

This keylogger script logs all keystrokes to a file named `keystrokes.txt`.

## Files

- `keylogger.py`: The main script that sets up the keylogger and logs keystrokes.
- `install_requirements.sh`: A shell script to install the necessary dependencies.

## Requirements

- Python 3
- Python 3 pip
- `keyboard` library

## Installation

1. Clone the repository or copy the keylogger script to your computer.
2. Make the installation script executable:
   ```bash
   chmod +x install_requirements.sh
   chmod +x keylogger.py
3. Install requirements:
   sudo ./install_requirements.sh
4. Run the program:
   sudo python3 keylogger.py
